Приклади використання зображень

Семантичний тег <figure> дозволяє групувати такий контент як зображення, ілюстрації, діаграми, фото та інше.
Тег <figcaption> використовується для підпису контенту всередині <figure>
Тег <img> дозволяє відображати зображення, атрибут src вказує на джерело зображення, alt - альтернативний опис зображення
Атрибути width та height дають можливість регулювати ширину та висоту зображення
