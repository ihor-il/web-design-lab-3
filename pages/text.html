Приклади використання тегів для роботи з текстом

  • <b> Текст жирним шрифтом
  • <strong> Текст жирним шрифтом (семантичний тег)
  • <em> Текст курсивом
  • <i> Текст курсивом (семантичний тег)
  • <small> Текст зменшеним шрифтом
  • <sub> Підпис
  • <sup> Надпис
  • <ins> Доданий текст
  • <del> Видалений текст
  • <mark> Виділений текст
  • <abbr> Семантичний тег, що вказує на акронім або абревіатуру - CSS (Cascade Style Sheets)
  • <address>
    Семантичний тег для вказання контактних даних
  • <bdo>
    Текст зліва направо
    Текст справа наліво
  • <blockqoute>
    Розширений цитатний блок. Lorem ipsum, dolor sit amet consectetur adipisicing elit. Error ut, id sint excepturi quae repellendus dolore sit consectetur delectus laborum blanditiis soluta sunt praesentium quos amet beatae officiis? Non, deserunt?
  • <cite> Тег що вказує на джерело цитати
  • <q> Невелика цитата
